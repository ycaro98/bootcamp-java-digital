O laço do/while (faça... enquanto), assim como o laço while, considera que enquanto uma determinada condição for válida o bloco de código será executado. Entretanto, do/while testa a condição após executar o código, sendo assim, mesmo que a condição seja considerada inválida no primeiro teste o bloco será executado pelo menos uma vez.

pseudocódigo:

//estrutura do controle do fluxo do while

do
{
	// comando que será executado até que a
	// expressão de validação torne-se falsa
}
while (expressão booleana de validação);
