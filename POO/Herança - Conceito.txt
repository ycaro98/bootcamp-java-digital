Conceituando: é o relacionamento entre classes em que uma classe chamada de subclasse(classe filha, classe derivada) é uma extensão, um subtipo, de outra classe chamada de superclasse(classe pai, classe mãe, classe base). Devido a isto, a subclasse consegue reaproveitar os atributos e métodos dela. Além dos que venham a ser herdados, a subclasse pode definir seus próprios membros.

Códigos:
Java				C#			Python
class A extends B {		class A : B {		class A(B):
	...				...		...
}				}
